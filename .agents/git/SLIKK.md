# git Skill

gitに関する操作を行う

## When to use this skill

- コミットメッセージの作成
- ブランチ作成

## How to use it

- コミットメッセージは日本語にすること
- コミットメッセージは接頭辞にfeat, fix, docs, style, refactor, test, choreなどをつける
- コミットは機能単位で行う
- ブランチはdevelopから作成
- ブランチ名は機能開発時はfeature/機能名、修正時はfix/修正内容、リリース時はrelease/リリース内容
