# Life Plan

人生の収支計画をシミュレーションするWebアプリ

## 基本ルール

- 常に日本語で回答する
- 質問には指定がない限り短い文章で回答
- 『鋭い指摘です』などの感想や相槌を省き、結論から簡潔に回答
- 長文のエラーは内容をそのまま出力せず重要な部分のみ出力し、エラーの原因や対処法のみ簡潔に伝える

## 技術スタック

- フレームワーク: React 19 + TypeScript + Vite
- UIライブラリ: MUI v9
- 状態管理: Zustand（persist middleware でlocalStorageに永続化）
- ルーティング: React Router v7

## 画面構成（ルーティング）

|パス|内容|
|:---|:---|
|/|ホーム|
|/family|家族構成の入力|
|/expense/home|住居費（賃貸 or 持ち家）|
|/expense/child|子ども費用（学校種別・ライフイベント）|
|/expense/car|車費用|
|/expense/living|生活費|
|/expense/other|その他費用|
|/income|収入入力|
|/results|シミュレーション結果（グラフ表示）|

## コーディングルール

- 基本的にMUIを使用する
- スタイルが必要な場合はsxで指定する。CSSファイルは作成しない
- 適切なコメントアウトをJsdoc形式で追加する
- 関数はアロー関数を使用して定義する
- ソース修正後は文法エラー・typeエラー・lintエラーが0であること
